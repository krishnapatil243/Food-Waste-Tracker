# EcoTrack - AI Food Waste Tracker

A modern web application that helps users track, reduce, and manage food waste effectively using AI-powered insights and a conversational chatbot assistant.

## Features

- **User Dashboard**: Track daily/weekly/monthly food waste with visual analytics
- **AI-Powered Chatbot**: Get personalized food storage tips, recipes from leftovers, and donation ideas powered by Google's Gemini AI
- **Waste Logging System**: Log wasted food items with quantity, reason, and type
- **Recipe Suggestions**: Discover creative ways to use ingredients before they spoil
- **Modern UI/UX**: Clean design with dark/light mode toggle and responsive layout
- **Gamification**: Earn badges for reducing waste and keeping streaks

## Technologies Used

- **Frontend**: HTML, Tailwind CSS, JavaScript
- **Charts**: Chart.js for data visualization
- **API Integration**: Compatible with free food/recipe APIs like Spoonacular
- **AI Integration**: Google's Gemini API for the intelligent chatbot assistant
- **Design**: Responsive layout with dark/light mode and smooth animations

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. The Gemini API key is already included in the application
